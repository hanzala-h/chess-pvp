# 🏆 Real-Time Chess PvP Game

Welcome to the Real-Time Chess PvP Game! This project is a multiplayer chess game built with modern web technologies to provide a smooth and engaging experience. Challenge your friends or play against random opponents in real-time!

## 🚀 Technologies Used

- **Node.js**: The backbone of our server-side logic.
- **Express.js**: Simplifying our routing and middleware.
- **MongoDB**: Storing game states and user data efficiently.
- **Socket.IO**: Enabling real-time, bi-directional communication between players.
- **EJS**: Rendering dynamic HTML for our views.

## 🌟 Features

- **Real-Time Gameplay**: Experience instantaneous moves and updates.
- **Multiplayer**: Play against friends or match with random opponents.
- **User Authentication**: Secure login and registration.
- **Game History**: View past games and moves.
- **Responsive Design**: Play on any device, whether it's a desktop, tablet, or mobile.

## 🛠️ Installation

Follow these steps to set up the project locally:

1. **Clone the repository**
    ```bash
    git clone https://github.com/yourusername/chess-pvp.git
    cd chess-pvp
    ```

2. **Install dependencies**
    ```bash
    npm install
    ```

3. **Set up MongoDB**
    - Make sure you have MongoDB installed and running.
    - Create a `.env` file in the root directory and add your MongoDB connection string:
      ```env
      MONGODB_URI=your_mongodb_connection_string
      DATABASE_NAME=your_database_name
      ```

4. **Run the application**
    ```bash
    npm start
    ```

5. **Visit the game in your browser**
   Open your browser and go to [http:localhost:3000](http:localhost:3000)
